Questions, answered.

What is World Monitor?
A real-time global intelligence dashboard. It fuses 500+ news feeds, conflict data, market prices, ship and flight tracking, satellite passes and cyber signals into one live map of the world, with AI analysis layered on top.
Is it really free?
Yes. The core dashboard — every map layer, every feed, all six monitors — is free with no signup. Pro adds the AI analyst, personal digests and API access for professionals.
Where does the data come from?
65+ providers including ACLED, UCDP, NASA FIRMS, USGS, IMF, BIS, Finnhub, FRED, AISStream and OpenSky, plus 500+ curated news feeds. Sources are cited on every panel.
Do I need an account?
No. The dashboard works instantly with no signup. An account only matters when you upgrade to Pro or want settings synced across devices.
What do I get with Pro?
WM Analyst (conversational AI across all live services), a personalized AI digest delivered daily, twice-daily or weekly to Slack, Discord, Telegram, Email or webhook, a custom widget builder, and MCP + API access — from $39.99/month.
Can AI agents like Claude or GPT use it?
Yes. World Monitor ships a 39-tool MCP server, so Claude, GPT or any MCP-compatible agent can query live geopolitical, market and infrastructure data as a tool.
Is it open source?
Yes — the platform is open source under AGPL-3.0 on GitHub. You can read the code, self-host, or build on it.
Is there a desktop app?
Yes. Native desktop builds for Windows, macOS and Linux are available, plus an Android TV app for wall displays.
